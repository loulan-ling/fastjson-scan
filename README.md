# fastjson-scan

快速批量扫描fastjson 漏洞

回显报错和DNSlog 批量检测fastjson

用法：
```
Usage of fastjson_scan:
  -f string
        url.txt
  -t int
        Thread default 10 (default 10)
  -u string
        http://192.168.1.1:80
```
